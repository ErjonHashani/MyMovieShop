Privacy Notice

Effective: | Front-End Demonstration Only

1. Browser Storage Usage

This demo uses browser storage technologies:

localStorage

  • • Persists until manually cleared
  • • Survives browser restarts
  • • Stores form inputs

sessionStorage

  • • Temporary per-tab storage
  • • Clears when tab closes
  • • Stores UI preferences

Cookies

  • • Not used in this demo
  • • No tracking implemented

2. Data Persistence

Any data you enter (forms, preferences etc.) will:

  • Remain in your browser after closing (localStorage)
  • Not expire automatically - stays until manually cleared
  • Never be transmitted to any server
⚠️

To completely remove all demo data, you must manually clear browser storage.

3. How To Clear Data

Chrome/Firefox/Edge:

  1. Press F12 for Developer Tools
  2. Go to Application → Storage
  3. Select Local Storage → Right-click domain → Clear

Complete Browser Cleanup:

  • Browser Settings → Privacy → Clear Browsing Data
  • Select "Cookies and other site data" + "Cached files"
  • Choose time range → Click "Clear data"

4. Your Privacy

This demo project:

  • Doesn't use analytics or tracking
  • Doesn't store personal information
  • Doesn't connect to any database
  • All code is client-side only
