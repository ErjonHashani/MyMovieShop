My Movie Shop Terms

This is a front-end development demonstration only

1. Project Nature

MyMovieShop is a static front-end demonstration created for portfolio/educational purposes. No real transactions or user accounts exist.

2. Browser Storage

This demo uses localStorage solely to:

  • Simulate form submission behavior
  • Maintain UI state during single sessions
  • Enable front-end functionality demonstrations

All data remains in your browser and is never transmitted externally.

3. No Data Collection

We do not collect, store, or process personal information. Any entered data exists only temporarily in your browser.

4. Clearing Data

To remove all demo data:

  1. Open browser developer tools (F12)
  2. Navigate to Application → Storage → Local Storage
  3. Right-click the domain and select "Clear"

5. Disclaimer

This is not a real service. Movie listings, prices, and other content are placeholder data.
